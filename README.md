# Fotogram – Photo Gallery

Eine interaktive Fotogalerie mit Miniaturansichten, Großansicht, und Tastatursteuerung.

## Features

- Dynamisch gerenderte Thumbnails für alle Alben
- Großansicht beim Klick auf ein Foto
- Navigation über Buttons und Pfeiltasten
- Schließen mit Hintergrundklick oder Esc-Taste
- Barrierefrei: Tabindex, Fokusmanagement und Screenreader-kompatibel
- Komprimierte Bilder für schnelles Laden
